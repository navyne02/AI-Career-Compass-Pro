# AI Career Compass Pro (Advanced AI Version)

This project implements **Hybrid Similarity** and **Deep Learning** for high-precision resume analysis.

---

### 🧠 Core Technologies

1. **SBERT (Sentence-BERT):** Deep Learning transformer model used for generating **semantic embeddings** to understand context.

2. **Hybrid Similarity:** Combines **Semantic Cosine Similarity** with **Ontology-based Lexical matching** for accurate scoring.

3. **Skill Ontology:** Hierarchical mapping of raw skills to specific **industry domains** (e.g., Python -> Data Science).

4. **Streamlit:** Advanced interactive UI for resume uploading and real-time analysis.
